桥接网络 🔗

hyperV管理器中增加一个虚拟交换机 youBridgeNmae 桥接到需桥接的外部网卡上，用文本编辑器打开 %USERPROFILE%\\wslconfig 文件或者用gvim的 e ~/wslconfig 打开wslconfig文件，添加如下内容

[wsl2]
networkingMode=bridged
vmSwitch=youBridgeNmae

启用systemd 🔗

在wsl2的linux shell中，修改/etc/wsl.conf(如果文件不存在则新建),增加

[boot]
systemd=true

wsl2中文修复 🔗

# 安装中文
sudo apt install language-pack-zh-hans
# locales选择en_US.UTF-8和zh_CN.UTF-8，默认值二选一即可
sudo dpkg-reconfigure locales
# 安装fontconfig
sudo apt install fontconfig

# 安装中文字体
cat >>/etc/fonts/local.conf<<EOF
<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
    <dir>/mnt/c/Windows/Fonts</dir>
</fontconfig>
<<EOF

# 刷新字体
sudo fc-cache -f -v
